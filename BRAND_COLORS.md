## Usage Guidelines

### Primary Orange (#FF9900)
**Use for:**
- Primary CTAs (Daftar, Mulai, Bayar)
- Important buttons
- Active states
- Brand highlights
- Links

**Don't use for:**
- Large backgrounds (too bright)
- Body text
- Subtle elements

### Secondary Blue
**Use for:**
- Secondary actions
- Info messages
- Trust indicators (verified, secure)
- Professional elements
- Navigation highlights

### Accent Teal
**Use for:**
- Success states
- Financial positive indicators
- Growth metrics
- Completed states
- Money-related elements

### Purple
**Use for:**
- Premium features
- Community elements
- Special badges
- Affiliate highlights
- VIP indicators
